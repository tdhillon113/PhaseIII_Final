# Binary Classification of Wine Quality and Multiclass Classification of Breast Cancer Subtype

## Step 1: Create and activate environment

```bash
conda create -n final_proj_env.yml
conda activate final_proj_env
```

## NOTE: make sure Breast_GSE45827.csv is downloaded and in the main directory of the folder in order to run the main.py file. If it is not, use the following link in order to download and access the csv file:

```https://www.kaggle.com/datasets/brunogrisci/breast-cancer-gene-expression-cumida```

## Step 2: Run the project

```bash
python main.py
```
[Warning: Binary Classification section will take a long time]
